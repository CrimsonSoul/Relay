Toolbar Integration

The enhance toggle sits at the end of the existing format toolbar. Toggle on = rules applied to preview. Toggle off = original text restored instantly. The original HTML is always preserved.

Enhancement Rules

Pattern Style Example
\d+\s*(days?|hours?|minutes?|weeks?) Yellow highlight 60 days, 2 hours
Dates (Month DD, YYYY / HH:MM UTC) Yellow highlight May 24, 2026, 14:15 UTC
must|required|immediately|do not|frozen Bold must, do not, All deployments are frozen
outage|failure|lockout|downtime|error Red highlight complete outage, account lockout
resolved|restored|operational|normal|lifted Green highlight fully resolved, operating normally
Percentages, ext. numbers, metrics Bold blue 100%, ext. 4357, 30 seconds
URLs / domains Blue underline password.company.com
Comma-separated lists (3+ items) Auto bullet list Payment API, Auth, Pipeline → <ul>
PascalCase / kebab-case / known infra terms Mono code pill us-east-1, API Gateway
