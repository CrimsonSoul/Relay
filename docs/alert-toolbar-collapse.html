Collapsible Highlight Palette

Three approaches for keeping the toolbar compact while still providing quick access to highlight colors.

Option A — Dropdown Popover RECOMMENDED
A single "Highlight" button with color preview dots. Click to open a popover with all swatches + labels + keyboard shortcuts. Collapses to one button width.

Option B — Active Color + Cycle Arrows
Shows only the currently selected highlight color as a single swatch. Click arrows to cycle through colors. Click the swatch to apply. Like the text color picker in Word/Google Docs.

Option C — Collapsible Second Row
Highlights live in a secondary toolbar row that slides open when you click a toggle or pen icon. Keeps the primary row clean. The row remembers its state.
