Interaction Model

  • Select text → click swatch — wraps selection in a <span data-hl="deadline"> with the matching style. Works exactly like Bold/Italic — select, click, done.
  • Click ✕ (clear) — removes any highlight from the selected text, unwrapping the span.
  • Manual highlights persist — they're stored in the body HTML itself, so they survive Compact/Enhance toggling. They're part of the "original" content.
  • Enhance auto-highlights don't clash — the enhance engine skips text that's already inside a data-hl span (manually highlighted). Manual always wins.
  • Compact preserves highlights — when compacting, any manually highlighted text is kept as-is. The compact engine only rewrites unhighlighted prose.
  • Sanitization — data-hl spans with known values (deadline, warning, success, number, service) are added to the existing sanitizer allowlist. Unknown values are stripped.
  • PNG capture — highlights render as inline styles on capture, so they appear correctly in the exported image and pasted Outlook emails.
